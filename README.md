# Handwritten Digit Recognition

A Convolutional Neural Network (CNN) built with TensorFlow and Keras that classifies handwritten digits with 99%+ accuracy.

# Technologies Used
1. Python 3.12
2. TensorFlow 2.21
3. Keras
4. NumPy
5. Matplotlib
6. Scikit-learn

# Model Architecture
 Conv2D → MaxPooling → Conv2D → MaxPooling → Flatten → Dense → Output

# Results
- Training Accuracy: 99.3%
- Test Accuracy: 99.17%

# How to Run
1. Clone the repo
2. Install dependencies: pip install -r requirements.txt
3. Run: python main.py
